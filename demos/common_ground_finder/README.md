## ğŸ¯ What Does This Do?

This demo teaches AI to be a **mediator** - someone who helps people agree. Instead of picking a winner, it:

1. **Listens** to two different viewpoints
2. **Identifies** the deep values behind each view
3. **Finds** shared concerns that both sides worry about
4. **Discovers** common goals everyone wants to achieve
5. **Suggests** a way to build a bridge between them

It's like having a super-smart friend who says, "Hey, you both want the same thing, just in different ways!"
