## Pattern Templates

### Analysis Pattern
1. Collect/describe input
2. Break into components
3. Analyze each component
4. Synthesize findings
5. Generate recommendations

### Creation Pattern
1. Define goal/constraints
2. Generate options
3. Evaluate options
4. Select best
5. Refine and finalize

### Problem-Solving Pattern
1. Define problem clearly
2. Identify constraints
3. Generate solutions
4. Evaluate solutions
5. Create action plan

### Learning Pattern
1. Introduce concept
2. Break into pieces
3. Explain each piece
4. Show connections
5. Apply to examples

### Refinement Pattern
1. Create initial version
2. Identify weaknesses
3. Suggest improvements
4. Apply improvements
5. Validate quality

---

Choose based on:
- **Complexity**: How many steps needed?
- **Creativity**: Divergent or convergent thinking?
- **Structure**: Defined process or exploratory?
- **Output**: Text, code, data, or decisions?
